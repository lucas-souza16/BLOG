Noção básicas sobre o Git

O git é um sistema de controle de versão que foi desenvolvido para facilitar o desenvolvimento de um projeto. Permitindo que duas ou mais pessoas trabalhem juntas. A funcionalidade desse sistema é juntar as partes do projeto de modo que cada membro de uma equipe faça uma parte e possa utilizar este sistema para juntar as partes do projeto. O conceito inicial é esse cada membro de uma equipe tem sua pasta do projeto trabalhado durante o dia e no final Git junta tudo em pasta no servidor principal evitando que esse arquivo seja sobrescrito, armazenando também o histórico e logs individuas de membros da equipe. Outra possibilidade que o Git oferece é de controlar o projeto e trabalhar de forma descentralizada sem precisar estar conectado ao servidor local.

Diferença entres os sistemas de controle de versão

O que deferência o git das outras ferramentas como as VCS (Subversion,Perforce,Bazar,etc.), esta na forma como o git trata os dados. Pois a maior parte dos outros sistemas que foram citados acima armazena informação como uma lista de mudanças por arquivos.
